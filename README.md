## â–¶ï¸ Running locally
 
   ```
   - Clone this repository:
   $ git clone https://github.com/yujisoyama/findyourduo.git
   
   - Enter in the folders project:
   $ cd findyourduo

   - For the backend:
   $ cd server
   $ npm install
   $ npm run dev

   - For the web app:
   $ cd web
   $ npm install
   $ npm run dev
   
   - For the mobile app:
   $ cd mobile
   $ npm install
   $ expo start
   
   - To run the frontend app (web or mobile) you need to have the server running.
  ```
